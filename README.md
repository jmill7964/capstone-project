# Automated Configuration Backup, Change, Verification, and Rollback

## Overview
This capstone project implements a safe network automation workflow using Python and Cisco IOS XE (CSR1000v). The lab is designed for undergraduate networking students and demonstrates how to perform configuration changes with proper safety controls, including backup, validation, and rollback.

## Automation Goal
To automate a configuration change while ensuring operational safety by:
- Backing up the current configuration
- Applying a controlled change
- Verifying the result
- Automatically rolling back if validation fails

## Target Audience
Undergraduate networking students learning network automation fundamentals.

## Tools and Technologies
- Python 3
- Netmiko (SSH-based network automation)
- Cisco IOS XE (CSR1000v in NDG Netlab DevNet environment)
- Linux terminal environment
